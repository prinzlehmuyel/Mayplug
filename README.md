# MayPlug

MayPlug is a vendor discovery and marketplace platform designed to connect users with local vendors, suppliers, manufacturers, and businesses.

## System Structure

- customer-site → public marketplace
- vendor-dashboard → vendor management system
- registry-admin → vendor onboarding/admin panel

## Backend

Firebase is used for:
- database
- realtime syncing
- vendor/product storage
- authentication

## Features

- Vendor registration
- Product listings
- Category browsing
- WhatsApp vendor connection
- Marketplace feed

## Goal

To create a structured local commerce ecosystem for easier vendor discovery and digital business visibility.
